В качестве платформы для создания приложения был выбран язык программирования R. К преимуществам R стоит отнести: 
1) свободное распространение в качестве программного обеспечения с использованием лицензии, позволяющей распространять ПО безо всяких ограничений в различных целях, в том числе и коммерческих; 
2) R предоставляет гибкие возможности по быстрому созданию аналитических инструментов (приложений) для решения определенного круга статистических задач и задач сбора и обработки данных.

Данное приложение позволит пользователю провести анализ данных жилой недвижимости, а именно посмотреть стоимость аренды жилья в зависимости от различных параметров, которые можно выбрать в ходе работы программы. Приложение может быть запущено как в режиме обычного пользователя, так и в режиме эксперта.

Описание структуры приложения:

Data: здесь пользователь будет складывать свои данные в различных форматах.

Graphics: сюда пользователь будет извлекать сформированные в приожении графику.

Library: каталог функций, которые будут вызываться операторами Scripts.

Notes: здесь пользователь сможет получить заметки и документацию.

Output: здесь пользователь будет получать возвращаемые и сохраненные данные.

Scripts: здесь будут располагаться управляющие файлы для работы с приложением.
